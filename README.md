# 🚀 Whitepace Landing Page Clone

A responsive frontend implementation of the **Whitepace SaaS Landing Page**, built using **HTML5** and **CSS3**. This project was created to strengthen my frontend development skills by recreating a modern, pixel-perfect landing page from a Figma Community design.

> **Note:** This project focuses on frontend implementation and responsive design. The original UI/UX design belongs to its respective creator on the Figma Community.

---

## ✨ Features

- 📱 Fully responsive layout
- 🧭 Responsive navigation bar with dropdown menus
- 🎯 Hero section with call-to-action buttons
- 📑 Multiple content sections with reusable layouts
- 💰 Pricing cards
- 💬 Testimonial section
- 🤝 Sponsors showcase
- 📄 Structured footer
- 🎨 Clean and modern UI
- ♻️ Reusable and organized code structure

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/dev-naresh608/whitepace-landing-page.git
```

2. Navigate to the project folder

```bash
cd whitepace-landing-page
```

3. Open `index.html` in your browser.

No additional dependencies or installation are required.
